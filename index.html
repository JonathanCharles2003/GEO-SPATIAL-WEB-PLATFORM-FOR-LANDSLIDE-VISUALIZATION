DEVELOPMENT OF AN INTEGRATED AND INTERACTIVE GEO-SPATIAL WEB PLATFORM FOR LANDSLIDE VISUALIZATION AND PREDICTION

The project "Development of an Integrated and Interactive Geo-Spatial Web Platform for Landslide Visualization and Prediction" aims to create a dynamic web-based solution for analyzing and predicting landslides using geospatial data. Utilizing HTML, CSS, and JavaScript along with the Leaflet library, the platform visually represents geospatial maps. GeoJSON files are employed to depict shape and boundary information, while GeoTIFF files display raster images, providing rich visual context. The platform incorporates interactive features such as checkboxes, allowing users to toggle between different types of GeoTIFF and GeoJSON files based on parameters like year, color, and other criteria. This flexibility enables the user to explore and compare geospatial data across various timeframes and conditions. Ultimately, the system integrates prediction functionality, supporting informed decision-making by forecasting landslide occurrences based on historical data and geospatial patterns. The platform is designed to enhance the visualization, analysis, and understanding of landslide risks in specific regions.
